# NumericRangeInput

El `NumericRangeInput` es un componente que nos permite especificar un rango de valores mínimo y máximo.

Los diferentes `props` que puede recibir son los siguietnes.

| **Prop**              | **Tipo**   | **Valor por defecto**  | **Descripción**                                                                         |
| :-------------------- | :--------- | :--------------------- | :-------------------------------------------------------------------------------------- |
| `label`               | `string`   | `undefined`            | Determina el `label` del input de rango                                                 |
| `minValuePlaceholder` | `string`   | `undefined`            | Especficia el `placeholder` del input de valor mínimo                                   |
| `maxValuePlaceholder` | `string`   | `undefined`            | Especficia el `placeholder` del input de valor máximo                                   |
| `onChange`            | `Function` | `(range: number[]) {}` | Recibe los valores mínimo y máximo en un `array` siendo el esquema general `[min, max]` |
