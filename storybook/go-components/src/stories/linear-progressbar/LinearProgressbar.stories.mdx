# LinearProgressBar

El `LinearProgressBar` es un componente gráfico para indicar el estado de un progreso lineal.

Los diferentes `props` que puede recibir son los siguietnes.

| **Prop**       | **Tipo**                          | **Valor por defecto** | **Descripción**                                                                                          |
| :------------- | :-------------------------------- | :-------------------- | :------------------------------------------------------------------------------------------------------- |
| `value`        | `number`                          | `undefined`           | Determina el valor actual de la barra de progreso                                                        |
| `totalValue`   | `number`                          | `undefined`           | Determina el valor TOTAL de la barra de progreso                                                         |
| `percentValue` | `number`                          | `undefined`           | En caso de no proporcionar los valores `value` y `totalValue` este prop determinará el valor de progreso |
| `color`        | ` lightblue` `blue` `lightgreen ` | `undefined`           | Determina el tipo de color que tendrá la barra de progreso                                               |
| `height`       | `number`                          | `1`                   | Determina la altura de la barra de progreso                                                              |
| `withLabel`    | `boolean`                         | `false`               | Determina si el componente saldrá con o sin `label`                                                      |
| `labelName`    | `string`                          | `undefined`           | Determina el valor del `label` en caso de que este aparezca                                              |
