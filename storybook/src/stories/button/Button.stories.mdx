# Button

The `Button` is a standard component.

Los diferentes `props` que puede recibir son los siguientes.

| **Prop**     | **Tipo**                                           | **Valor por defecto** | **Descripción**                               |
| :----------- | :------------------------------------------------- | :-------------------- | :-------------------------------------------- |
| `textButton` | `string`                                           | `undefined`           | Determina el texto que se muestra en el botón |
| `color`      | `blue` `gray` `transparent` `transparent-outlined` | `blue`                | Determina el color y demás estilos del botón  |
| `css`        | `string`                                           | `undefined`           | Permite añadir estilos adicionales de CSS     |
| `disabled`   | `boolean`                                          | `false`               | Determina si el botón estará deshabilitado    |
| `iconButton` | `React.ReactNode`                                  | `undefined`           | Determina el componente icono del botón       |
| `size`       | `l` `m` `s`                                        | `m`                   | Determina el tamaño del botón                 |
| `ctaButton`  | `boolean`                                          | `false`               | Determina si el botón será de tipo `cta`      |
